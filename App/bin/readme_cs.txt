Co je OmegaT?
===============

OmegaT je volný, open source multiplatformový nástroj na překlad podporovaný počítačem (Computer Assisted Translation) s vyhledáváním přibližných překladů, překladovou pamětí, vyhledáváním klíčových slov, používá vaše vlastní glosáře a využívá překlady pro aktualizované projekty.

Co potřebujete k instalaci OmegaT
===================================

OmegaT požaduje k instalaci na váš systém prostředí Java Runtime Environment (JRE) ve verzi 1.8 a novější.

Doporučujeme použít instalační balíček OmegaT, který obsahuje JRE, a tak si ušetříte možné potíže s výběrem, získáním a instalací programu.



Instalace programu OmegaT Windows
===========================

Spusťte instalační program.



Instalace OmegaT (Mac)
=======================

Rozbalte archív OmegaT.zip a získáte tak adresář, který obsahuje soubor s dokumentací a aplikací OmegaT. Přesuňte adresář na vhodné místo jako je adresář s aplikacemi.



Instalace OmegaT Linux
=========================

Umístěte archív do jakéhokoliv vhodného adresáře a rozbalte jej. OmegaT je připravena ke spuštění.

Nicméně úhlednější a přívětivější instalaci spustíte použitím instalačního skriptu (linux-install.sh). Abyste spustili tento skript, otevřete okno terminálu (konzole), změňte aktivní adresář na adresář, který obsahuje OmegaT.jar a skript linux-install.sh, a spusťe skript příkazem ./linux-install.sh. 



Instalace OmegaT (Solaris, FreeBSD, ...)
=========================================

Umístěte archív do jakéhokoliv vhodného adresáře a rozbalte jej. OmegaT je připravena ke spuštění.



Použití Java Web Start pro instalaci OmegaT (všechny platformy)
===========================================================

Pokud již máte na svém systému nainstalovanou Javu, pak se jedna z možností instalace OmegaT nabízí použití Java Web Start.

V takovém případě stačí stáhnout následující soubor a spustit jej:

   https://omegat.sourceforge.io/webstart/OmegaT.jnlp

Při prvním spuštění bude nainstalováno nejen prostředí přesně pro váš počítač ale také samotná aplikace. Při dalším spuštění již nemusíte být online.



Spuštění OmegaT (Windows)
==========================

Pokud jste během instalace vytvořili odkaz na ploše, stačí dvojklik na tento odkaz.

Instalační program může vytvořit zkratky v menu start, na ploše a v panelu rychlého spuštění. Stejně tak můžete ručně přetáhnout soubor OmegaT.exe do menu Start, na Plochu nebo na panel Snadné spuštění.

Pokud ve vašem Správci souborů vidíte soubor OmegaT a ne OmegaT.exe (Průzkumník Windows), změňte nastavení tak, aby byly zobrazovány přípony souborů.



Spuštění OmegaT (Mac)
======================

Dvojklik na aplikaci OmegaT.

Můžete přetáhnout aplikaci OmegaT na dock nebo lištu nástrojů v okně Finderu a umožnit tak spuštění OmegaT z libovolného umístění. Také můžete spouštět program z políčka vyhledávání Spotlight.



Spuštění OmegaT (Linux)
========================

Pokud jste použili skript linux-install.sh, pak bude možno spustit OmegaT takto:

  Alt+F2

a pak:

  O aplikaci OmegaT

Přívětivější způsob spouštění OmegaT nabízí skript Kaptain (omegat.kaptn), který je součástí staženého balíčku. Aby šlo tento skript spouštět, je nutno první instalovat Kaptain. Pak můžete spustit spouštěcí skript Kaptain pomocí:

  Alt+F2

a pak:

  omegat.kaptn



Spuštění OmegaT z příkazového řádku (všechny systémy)
====================================================

Příkaz ke spuštění OmegaT je:

cd <adresář, kde je uložený soubor OmegaT.jar>

<jméno a cesta k souboru spustitelného Javou> -jar OmegaT.jar

(Soubor spustitelný Javou je soubor „java“ v Linuxu a „java.exe“ ve Windows.
Pokud je Java instalována na úrovni systému, a když je také zadaná cesta v příkazu pro spouštění, nemusíte vkládat úplnou cestu.)
