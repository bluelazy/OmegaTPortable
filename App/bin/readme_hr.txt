Što je OmegaT?
===============

OmegaT je besplatan i „slobodan” višeplatformski alat otvorenoga kôda za računalno potpomognuto prevođenje koji omogućava korištenje djelomičnih podudaranja, prijevodnih memorija, pretraživanja po ključnim riječima, glosara i prevedenih sadržaja u ažuriranju projekata.

Preduvjeti za instaliranje programa OmegaT
==========================================

Za korištenje programa OmegaT morate na računalu imati instaliran Java Runtime Environment (JRE) inačice 1.8 ili novije.

Preporučujemo instalacijske pakete OmegaT koji obuhvaćaju JRE, čime ćete biti pošteđeni brige oko njegovoga odabira, preuzimanja i instaliranja.



Instaliranje programa OmegaT (Windows)
======================================

Pokrenite instalacijski program.



Instaliranje programa OmegaT (Mac)
==================================

Raspakirajte arhivu .zip programa OmegaT, čime se izrađuje mapa s datotekom dokumentacije i programom. Premjestite mapu na odgovarajuću lokaciju, primjerice u mapu Aplikacije.



Instaliranje programa OmegaT (Linux)
====================================

Premjestite arhivu u bilo koju odgovarajuću mapu pa je raspakirajte. OmegaT je potom spremna za pokretanje.

Međutim, instalaciju možete praktičnije i jednostavnije obaviti putem instalacijske skripte (linux-install.sh). Skriptu možete iskoristiti tako da otvorite prozor terminala (konzole), upišete mapu u kojoj se nalaze datoteka OmegaT.jar i skripta linux-install.sh, nakon čega izvršavate skriptu naredbom ./linux-install.sh.



Instaliranje programa OmegaT (Solaris, FreeBSD...)
==================================================

Premjestite arhivu u bilo koju odgovarajuću mapu pa je raspakirajte. OmegaT je potom spremna za pokretanje.



Korištenje tehnologije Java Web Start za instaliranje programa OmegaT (sve platforme)
=====================================================================================

Tehnologija Java Web Start je jedan od načina za instaliranje programa OmegaT, pod uvjetom da na računalu već imate instaliran programski jezik Java.

U tu svrhu trebate preuzeti pa izvršiti sljedeću datoteku:

   https://omegat.sourceforge.io/webstart/OmegaT.jnlp

Njome se prilikom prvoga pokretanja instalira valjano okruženje koje odgovara i vašemu računalu i samome programu. Naknadni pozivi programa ne moraju biti obavljani putem weba.



Pokretanje programa OmegaT (Windows)
====================================

Ako ste tijekom instalacije izradili prečac na radnoj površini, pritisnite ga dvaput lijevom tipkom miša.

Instalacijskim je programom moguće izraditi i prečace na izborniku Start, na radnoj površini, kao i na traci za brzo pokretanje. Također možete odvući datoteku OmegaT.exe na izbornik Start, radnu površinu, ili na traku za brzo pokretanje.

Ako u programu za upravljanje datotekama (Windows Explorer) vidite datoteku OmegaT, ali ne i OmegaT.exe, promijenite njegove postavke tako da prikazuje i datotečne nastavke.



Pokretanje programa OmegaT (Mac)
================================

Dvaput pritisnite datoteku programa OmegaT.

Možete odvući datoteku programa OmegaT na Dock ili alatnu traku prozora Finder kako biste program mogli pokretati s bilo koje lokacije. Također ga možete pokretati i iz polja za pretraživanja Spotlighta.



Pokretanje programa OmegaT (Linux)
==================================

Ako ste koristili skriptu linux-install.sh, program OmegaT biste morali biti u stanju pokrenuti putem:

  Alt+F2

a zatim:

  omegat

Program OmegaT možete pokretati na jednostavniji način putem skripte Kaptain iz programskoga paketa (omegat.kaptn). Da biste mogli koristiti skriptu, morate instalirati Kaptain. Zatim možete aktivirati skriptu Kaptain za pokretanje putem:

  Alt+F2

a zatim:

  omegat.kaptn



Pokretanje programa OmegaT naredbenim retkom (svi sustavi)
==========================================================

Naredba za pokretanje programa OmegaT glasi:

cd <mapa u kojoj se nalazi datoteka OmegaT.jar>

<naziv i putanja do izvršne datoteke Java> -jar OmegaT.jar

(Izvršna datoteka programskoga jezika Java jest datoteka java u sustavu Linux ili java.exe u sustavu Windows.
Ako je Java instalirana na razini sustava i nalazi se na putanji naredbe, nije potrebno upisivati punu putanju.)
