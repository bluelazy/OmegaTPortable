Користувачі Linux (Intel)

Який пакет завантажувати?

Чи встановлена у Вас версія Java, сумісна з SUN 1.4 JRE?

Так: 

завантажте OmegaT_1.8.X_Without_JRE.zip. Цей пакет можна використовувати на будь-якій платформі зі встановленою версію Java 1.4 JRE.

Ні / не знаю:

завантажте OmegaT_1.8.X_Linux.tar.gz. Цей пакет містить SUN's Java Runtime Environment. Ця версія Java не конфліктує з іншими інсталяціями Java, встановленими на Вашій системі.

Встановлення OmegaT

Щоб встановити OmegaT, просто розпакуйте завантажений файл. У поточному каталозі створиться каталог omegat/, де Ви знайдете всі необхідні файли для запуску OmegaT. Щоб розпакувати файл .tar.gz:

$ tar xf завантажений_файл.tar.gz

Додавання OmegaT в меню (KDE) або панелі (Gnome)

Користувачі KDE

Ви можете додати OmegaT до Вашого меню наступним чином:

  • Центр керування - Робочий стіл - Панелі - Меню - Редагувати меню K - Файл - Новий елемент/Нове підменю

  • Після вибору необхідного меню, додайте підменю/елемент, натиснувши Файл - Нове підменю і Файл - Новий елемент. Введіть „OmegaT“ як назву нового елемента.

  • У полі „Команда“ використайте кнопку навігації, щоб знайти скрипт запуску OmegaT (файл OmegaT у розпакованому каталозі), і виберіть цей файл.

  • Натисніть на кнопці іконки (справа від полів Імʼя/Опис/Коментар) - Інші іконки - Огляд, і перейдіть до підкаталогу /images в каталозі OmegaT. Виберіть іконку OmegaT.png.

  • Збережіть зміни, натиснувши Файл – Зберегти.

Користувачі Gnome

Ви можете додати OmegaT до Вашої панелі (меню вгорі екрана) наступним чином:

  • Натисніть правою кнопкою миші на панелі - Додати новий лаунчер.

  • Введіть „OmegaT“ у поле „Імʼя“; у полі „Команди“ використайте кнопку навігації, щоб знайти скрипт запуску OmegaT (файл з назвою „OmegaT“ у нерозпакованому каталозі). Виберіть і натисніть OK.

Запуск OmegaT

Після встановлення OmegaT Ви можете запускати програму відразу з командного рядка, створити скрипт з параметрами запуску або натиснути мишкою на файл OmegaT.jar, щоб відразу запустити OmegaT. Методи залежать від дистрибутива. Переконайтеся, що налаштування PFAD правильні й файли .jar асоційовані з лаунчером Java.

Користувачі Windows

Який пакет завантажувати?

Чи встановлена у Вас версія Java, сумісна з SUN 1.4 JRE?

Так:

завантажте файл OmegaT_1.8.X_Windows_without_JRE.exe. Цей пакет можна використовувати на будь-якій платформі зі встановленою версію Java 1.4 JRE.

Ні / не знаю:

завантажте OmegaT_1.8.X_Windows.exe. Цей пакет містить SUN's Java Runtime Environment. Ця версія Java не конфліктує з іншими інсталяціями Java, встановленими на Вашій системі.

Встановлення OmegaT

Щоб установити OmegaT, натисніть двічі на завантаженій програмі та дотримуйтесь інструкцій.

Додавання OmegaT в стартове меню або на робочий стіл.

Під час встановлення програма запитає, чи потрібно створити підкаталог в меню „Пуск“ і ярлики на робочому столі та на панелі швидкого запуску.

Ярлики можна створити пізніше, перетягнувши OmegaT.exe на робочий стіл або в меню „Пуск“.

Запуск OmegaT

Після встановлення OmegaT Ви можете відразу запустити програму, натиснувши на файлі OmegaT.jar або з командного рядка.

Але найпростіше — запустити файл „OmegaT.exe“. У цьому випадку параметри будуть прочитані з файлу OmegaT.l4J.ini, який знаходиться у тому ж каталозі, що й цей файл .exe і Ви можете відредагувати його під власні налаштування. Такий файл INI резервує 1GB памʼяті, встановлює французьку мовою користувача і Португалію країною мови:

# OmegaT.exe runtime configuration
#
# To use a parameter, remove the '#' before the '-'
# Memory
-Xmx1024M
# Language
-Duser.language=FR
# Country
-Duser.country=PT
# Anti-aliasing (Java 1.4 and 1.5 only)
#-Dswing.aatext=true
# Entity expansion for XML (Java 1.4 only)
#-DentityExpansionLimit=400000

Користувачі MacOSX

Який пакет завантажувати?

Завантажте OmegaT_1.8.X_MacOSX.zip.

Встановлення OmegaT

Двічі натисніть на файл OmegaT_1.8.X_MacOSX.zip, щоб розпакувати його. Образ диска OSX (OmegaT. dmg) змонтує себе і покаже два файли: documentation.html і OmegaT. app. Скопіюйте ці два файли в необхідний каталог (наприклад, Applications, в якому ви можете створити підкаталог OmegaT). Після виконання цих дій Ви можете видалити файл OmegaT_1.8.X_MacOSX.zip.

Як додати OmegaT в Док

Ви можете додати OmegaT в Док, перетягнувши туди файл OmegaT.app.

Запуск OmegaT

Щоб запустити OmegaT, двічі натисніть на файл OmegaT.app або на його ярлик в Доку.

Якщо Ви хочете змінити параметри запуску Java з командного рядка, рекомендується використовувати пакет, доступний для „інших систем“, див. нижче. Також можна напряму змінювати файл Info.plist, який знаходиться в пакеті OmegaT.app.

Інші системи

Який пакет завантажувати?

OmegaT доступний для платформ Linux (Intel x86) і Windows зі встановленим SUN Java JRE. Користувачі інших платформ (Linux PowerPC, Linux x64, Solaris SPARC/x86/x64, Windows x64 тощо) повинні мати на своїх ОС сумісну версію JAVA JRE, щоб використовувати OmegaT.

Чи встановлена у Вас версія Java, сумісна з SUN 1.4 JRE?

Так:

завантажте OmegaT_1.8.X_Without_JRE.zip. Цей пакет можна використовувати на будь-якій платформі зі встановленою версію Java 1.4 JRE.

Не знаю:

відкрийте термінал і наберіть „java -version“. Якщо зʼявиться „Команда не знайдена“ або схоже повідомлення, скоріш за все на Вашій системі не встановлена Java.

Ні:
встановіть Java JRE для Вашої системи (див. нижче) і завантажте OmegaT_1.7.3_Without_JRE.zip. SUN надає JRE для Solaris SPARC/x86 (Java 1.5) і для Linux x64, Solaris x64, Windows x64 (Java 1.5): http://java.sun.com/products/archive/. IBM надає версії JRE для Linux PowerPC:
http://www.ibm.com/developerworks/java/jdk/linux/download.html
Дотримуйтесь інструкцій встановлення необхідного Вам пакета.

Встановлення OmegaT

Щоб встановити OmegaT, просто розпакуйте файл OmegaT_1.8.X_Without_JRE.zip. Буде створений каталог ./OmegaT_1.8.X_Without_JRE/, в якому знаходяться всі необхідні файли для запуску OmegaT.

Встановлення практичних комбінацій клавіш

Достримуйтесь інструкцій Вашої системи, щоб установити ярлики OmegaT у відповідних місцях.

Запуск OmegaT

Після встановлення OmegaT Ви можете запускати програму відразу з командного рядка, створити скрипт з параметрами запуску або натиснути мишкою на файл OmegaT.jar, щоб відразу запустити OmegaT. Методи залежать від дистрибутива. Переконайтеся, що налаштування PFAD правильні й файли .jar асоційовані з лаунчером Java.

Запуск з командного рядка

Зазвичай OmegaT не потрібно запускати з командного рядка. Проте метод запуску з командного рядка дає можливість запускати OmegaT і контролювати аспекти поведінки програми. Існує два способи запуску OmegaT з командного рядка.

Користувачам пакета Mac OSX, які хочуть контролювати OmegaT з командного рядка, рекомендовано використовувати пакет, доступний для „інших систем“. Дивіться вище.

Спосіб 1: відкриття вікна командного рядка

Вікно командного рядка називається також „термінальне вікно“. У Windows воно називається „Вікно MS-DOS“ і відкривається в меню „Пуск - Усі програми - MS-DOS“. Відповідний додаток для Mac OSX називається „Terminal“, і знаходиться в „Усі програми → Службові програми“.

Зазвичай для запуску OmegaT потрібно вказати дві наступні команди. Найперше:

cd {каталог}

де {каталог} —це назва каталогу зі вказаним повним шляхом, де знаходиться програма OmegaT, а саме файл OmegaT.jar. Тому ця команда, скоріш за все, виглядатиме так:

У Windows
cd C:\Program Files\OmegaT
У Mac OSX
cd /Applications/OmegaT
У Linux
cd /usr/local/omegat

Ця команда змінює каталог на той, де знаходиться активний файл OmegaT. Друга команда запускає OmegaT. У найпростішому вигляді вона така:

java -jar OmegaT.jar

Зверніть увагу на реєстр літер — це важливо.

Ці способи особливо зручні для пошуку помилок, якщо такі зʼвляються під час роботи програми. Повідомлення про помилку знаходиться у термінальному вікні й може містити корисну інформацію про причину виникнення помилки.

Спосіб 2: змінення файлу .bat/ скрипту запуску

Запускати програму постійно способом 1 дещо непрактично. Тому дві вище описані команди були поєднані в одному файлі (скрипт, також названий „файл .bat“ у системах Windows).

Якщо запустити цей файл, то команди, які він містить, виконаються автоматично. Відповідно, щоб внести зміни до команди запуску, достатньо змінити лише цей файл. Якщо OmegaT запускається з файлу скрипту, нові команди виконуються.

Аргументи команди запуску

Основна команда вже була згадана вище. Зміни в цій команді передбачають додавання „аргументів“. Аргументи додаються після початкової „java“ і перед частиною „-jar OmegaT.jar“. Зверніть увагу: у Windows Ви можете налаштувати файл OmegaT.l4J.ini на свій розсуд.

Нижче наведений список можливих аргументів. Ви можете отримати більше інформації про аргументи, якщо введете man java у термінальному вікні. man — це команда, яка показує посібник з вибраної команди. Можливо, у Вашій системі є посібник команди java, написаний Вашою мовою.

Мова інтерфейсу користувача: -Duser.language=XX

     

Зазвичай, а саме при запуску OmegaT без будь-яких аргументів, програма спочатку визначає мову операційної системи користувача. Якщо інтерфейс користувача може відображатися цією мовою, OmegaT її використає. Це означає, що якщо операційна система користувача російською мовою і OmegaT локалізована російською, то в програмі інтерфейс користувача, меню тощо будуть цією ж мовою. Якщо мова системи користувача не підтримується, буде використана англійська мова. Це стандартна поведінка.

Аргумент „-Duser.language=XX“ вказує OmegaT замість мови операційної системи використовувати вказану мову. "XX" в команді означає код бажаної мови (дві літери). Щоб запустити OmegaT з інтерфейсом французькою мовою (наприклад, на російській операційній системі), потрібна команда: java -Duser.language=fr -jar OmegaT.jar

Країна користувача: -Duser.country=XX

     

Окрім мови Ви можете також вказати країну, наприклад CN або TW для китайської мови. Щоб вікрити коректний посібник швидкого запуску, потрібно вказати як мову, так і країну. Це необхідно, навіть якщо існує лише одна комбінація, наприклад, pt_BR у випадку Португальська / Бразилія.

Згладжування шрифтів: -Dswing.aatext=true

     

Цей аргумент вмикає згладжування шрифтів, таким чином покращує їхній зовнішній вигляд.

Виділення памʼяті: -XmxZZM

     

Ця команда виділяє OmegaT більше памʼяті. За замовчуванням програмі виділено 256 MB, тому немає сенсу виділяти менше памʼяті:. "ZZ" означає виділену кількість памʼяті в мегабайтах. Тому, щоб запустити OmegaT з 1024 Mb (1 гігабайт) памʼяті, потрібно вказати наступну команду: java -Xmx1024M -jar OmegaT.jar

Аргументи можна комбінувати: щоб запустити OmegaT з усіма вище вказаними прикладами, команда повинна виглядати так:
java -Dswing.aatext=true -Duser.language=pt -Duser.country=BR -Xmx1024M -jar OmegaT.jar

Створення OmegaT з джерела

Який пакет завантажувати?

Завантажте OmegaT_1.8.X_Source.zip
Цей пакет містить файли з джерельним кодом відповідної версії програми. Його можна використовувати на будь-якій платформі зі встановленою версію Java 1.4 JDK або сумісною Java-системою. Крім того, Вам буде необхідна утиліта ant, щоб зібрати OmegaT.
OmegaT використовує SubVersion для контролю версій. Найновіший код знаходиться на сервері SVN Server:
https://omegat.svn.sourceforge.net/svnroot/omegat/trunk

Створення OmegaT

Розпакуйте файл OmegaT_1.8.X_Source.zip і перейдіть в каталог OmegaT_1.8.X_Source або в каталог ./omegat/, якщо Ви скачали перевірений код з SVN. Переконайтеся, що в каталозі знаходиться файл build.xml. Потім введіть в командному рядку наступну команду:

$ ant jar release

Вона створить повний пакет OmegaT в каталозі ./dist/ з усіма необхідними файлами для запуску програми.
