1. Ιστότοποι του OmegaT και το OmegaT στο έργο SourceForge

Ο ιστότοπος του OmegaT περιέχει συνδέσμους με πολυάριθμους πόρους για το OmegaT. Υποστήριξη χρηστών του OmegaT προσφέρεται σε εθελοντική βάση στο Yahoo! Ομάδα χρηστών. Τα FAQ (Συνηθέστερες ερωτήσεις) αποτελούν ένα καλό σημείο εκκίνησης για να βρείτε απαντήσεις στις ερωτήσεις που, ίσως, έχετε. Για την πιο πρόσφατη έκδοση του OmegaT, ανατρέξτε στη σελίδα downloads, στη διεύθυνση www.omegat.org. Μπορείτε, επίσης, να υποβάλετε Αναφορά σφαλμάτων και Ατιήματα βελτιώσεων.

2. Αναφορά σφαλμάτων

Να θυμάστε ότι μια αναφορά σφάλματος, για να είναι καλή, πρέπει να περιέχει τρία πράγματα:

  • Τα απαραίτητα βήμα για την αναπαραγωγή του

  • Τι θα περιμένατε, κανονικά, να δείτε

  • Τι είδατε, αντιθέτως

Μπορείτε να βρείτε περισσότερα στο άρθρο Ανώδυνη Αναφορά Σφαλμάτων του Joel Spolsky.

Θα πρέπει να επισυνάψετε αντίγραφα αρχείων, τμήματα του καταλόγου καταγραφής (log), στιγμιότυποα οθόνης και οτιδήποτε άλλο που θεωρείτε ότι θα βοηθήσει τους προγραμματιστές να εντοπίσουν και να διορθώσουν το σφάλμα. Σημειώστε ότι οι αναφορές σφαλμάτων και τα αιτήματα βελτιώσεων είναι όλα δημοσίως ορατά, συνεπώς, δεν θα πρέπει να προσθέσετε οποιαδήποτε ευαίσθητα αρχεία. Αν θέλετε να καταγράψετε στην αναφορά σας τι ακριβώς συμβαίνει, εγγραφείτε ως χρήστης στο SourceForge, συνδεθείτε και καταχωρήστε μια αναφορά σφάλματος ή, απλώς, κάντε κλικ στην Παρακολούθηση, στο πάνω μέρος της αναφοράς.

3. Έργο συνεισφορών στο OmegaT

Για να συνεισφέρετε στο OmegaT:

Κατ' αρχήν, εγγραφείτε στην Ομάδα χρηστών (μέσω web, ή στέλνοντας ένα e-mail στο OmegaT-subscribe@yahoogroups.com). Για να συμμετάσχετε στην προσπάθεια ανάπτυξης του OmegaT μπορείτε να ενώσετε τις δυνάμεις σας με την ομάδα ανάπτυξης, μέσω του Web, ή στέλνοντας μήνυμα e-mail στο omegat-development-request@lists.sourceforge.net βάζοντας ως θέμα μηνύματος τη λέξη "subscribe".

Για να μεταφράσετε τη Διεπαφή χρήστη του OmegaT, το Εγχειρίδιο χρήστη, ή άλλα συναφή έγγραφα:

Πρώτα από όλα, διαβάστε τον Οδηγό Τοπικοποίηση και Παραμετροποίηση του OmegaT και εγγραφείτε στην ομάδα μεταφραστών του OmegaT μέσω του Web, ή στέλνοντας ένα e-mail στο omegat-l10n-request@lists.sourceforge.net βάζοντας ως θέμα τη λέξη "subscribe".

Για να προσφέρετε οικονομική υποστήριξη στο έργο OmegaT:

Αν θέλετε να βοηθήσετε στη συνέχιση της ανάπτυξης του OmegaT, θα εκτιμηθεί πάρα πολύ - κάντε κλικ σε αυτό τον σύνδεσμο, για να μεταβείτε στο Λογαριασμός OmegaT στο PayPal .
