1. Como discargar e installar le dictionarios

Le dictionarios in OmegaT es basate sur le formato StarDict o sur le Lingvo DSL. Pro installar le files necessari in le formato StarDict, procede como seque:

  • Cerca pro le combination de linguas requirite - per exemplo sur le StarDict Platform ante.

  • Discarga le file - isto debe esser un archivo tarball (extension tar.bz or tar.bz2).

  • Usa le utilitate untar (o su equivalente, per exemplo winrar in Windows) pro extraher su contentos in le plica "Dictionary" del projecto. Ibi debe esser tres files, con extensiones dz, idx e ifo.

Nota que in plus al dictionarios "fonte-destination" tu pote obtener, con le function Dictionario, accesso a informationes como:

  • Dictionario Webster Revised Unabridged (1913)

  • Dictionario del anglese contemporanee Longman

  • Le Concise Encyclopedia Britannic

  • et cetera

Alicun del dictionarios non ha filas attaccate - id es lor uso es "gratuite", e altere, como le selection supra, es sub Le licentia GPL. Le exemplo sequente monstra le dictionario Merriam Webster 10th "in action":
